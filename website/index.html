Tunecamp

A modern static site generator for musicians and music labels.

Point Tunecamp at a folder hierarchy containing your audio files.
Within minutes, Tunecamp builds a complete website presenting your work.
The resulting site requires no database, no programming, and minimal maintenance.

Audio-First

Built specifically for musicians and music labels with automatic metadata extraction.

  • Multi-format Support MP3, FLAC, OGG, WAV, M4A, OPUS and more
  • Automatic Metadata Reads tags from your audio files
  • Built-in Player Modern HTML5 audio player with streaming support
  • Remote Audio Files Use external URLs for audio (CDN, storage services)
  • Zero Database Pure static HTML generation

Download Models

Flexible download options to match your distribution style.

  • Free Downloads Make all tracks immediately available
  • Soft Paycurtain (Honor System) Show PayPal/Stripe links with a suggested price, but downloads remain always available - it's up to the listener to support you
  • Unlock Codes Protect downloads with codes validated via GunDB (requires self-hosting to generate codes)
  • Streaming Only No downloads, audio streaming only (perfect for distribution platforms)
  • Download Stats Track download counts via decentralized GunDB

Quick Start

# Install Tunecamp
npm install -g tunecamp

# Create your music catalog
mkdir my-music
cd my-music

# Initialize catalog structure
tunecamp init .

# Add your audio files
cp *.mp3 releases/example-album/tracks/
# (optional) rename example-album to your release slug

# Configure your catalog
# Edit catalog.yaml, artist.yaml, and release.yaml files

# Generate your website
tunecamp build . --output ./public

# Deploy to any static host
# Upload the public folder to Netlify, Vercel, GitHub Pages, etc.
