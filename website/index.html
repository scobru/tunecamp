Tunecamp

A static site generator for audio producers.

Point Tunecamp to a folder hierarchy containing your audio files.
Within minutes, Tunecamp builds a complete website presenting your work.
The resulting site requires no database, no programming, no maintenance, and is compatible with virtually every webhost on this planet.

Streaming

Your audio files are automatically transcoded for streaming on various channels.

  • Web Player Visitors can listen directly on your tunecamp site
  • Podcast RSS Let your audience subscribe through their podcasting app
  • M3U Playlists Allow fans to stream to various media players and devices

Downloads

Automatically generated downloads in your chosen formats, as single tracks or full release archives.

  • Unlock Codes Provide exclusive access to downloads
  • Soft Paycurtain Ask financial support on third-party platforms
  • External Downloads Link to third-party platforms for download/purchase

Quick Start

# Install Tunecamp
npm install -g tunecamp

# Create your music catalog
mkdir my-music
cd my-music

# Add your audio files
mkdir releases/my-album
cp *.mp3 releases/my-album/

# Generate your website
tunecamp build . --output ./public

# Deploy to any static host
# Upload the public folder to Netlify, Vercel, GitHub Pages, etc.
